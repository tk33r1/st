最適化の仕組み

このツールは、ブラウザの処理能力を活用してファイルを高速かつ安全に最適化します。ファイルが外部サーバーに送信されることはありません。

PNG/JPEG画像の変換

画像ファイルがアップロードされた場合、自動的に輪郭を解析してベクターデータ（SVG）に変換します。この処理には imagetracerjs を使用しています。変換後、さらにSVGとして極限まで圧縮されます。

  • IllustratorやSketchなどが残す不要なメタデータを削除します
  • 非表示の要素や空の属性を安全に削除します
  • パスデータやカラーコードの記述を最小化します（SVGOを使用）

アグレッシブ圧縮について

これをオンにすると、小数点以下の精度を極限まで削り、viewBoxなどを強制削除してさらなる軽量化を図ります。ただし、一部の複雑な図形では表示が崩れる可能性があります。

Data URI 変換について

これをオンにすると、最適化されたSVGを直接CSSの background-image や HTMLの img src に貼り付けられるBase64形式（data:image/svg+xml;base64,...）で出力します。
