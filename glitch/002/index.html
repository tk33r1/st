Apple アカウントをトルコで作って
サブスク料金を劇的に下げる方法

● Shinya Takeda 2026.05.01 ⏱ 約7分
image here
⚠ 免責事項： Apple の利用規約では、居住国と異なる地域の App Store を使用することを禁じている場合があります。 アカウント停止リスクを理解した上で、自己責任で実行してください。為替レートにより節約額は変動します。

なぜトルコなのか

Apple は国ごとに価格を設定している。App Store や iCloud+、Apple Music、Apple TV+ などのサブスクリプション料金は、 各国の経済水準・物価・為替レートをもとに決まる。 トルコリラは長期的なインフレにより大幅に下落しており、 結果としてトルコの Apple サブスク価格は日本の5〜10分の1以下になっている。

必要なもの

CHECKLIST: 別のメールアドレス（Gmailなど）／ トルコの住所（後述）／ VPN（任意だが推奨）／ トルコ対応の支払い手段（後述）

手順

新しい Apple ID 用メールを用意する

既存の Apple ID とは別のメールアドレスを用意する。Gmail や iCloud の「エイリアス」でも可。これがトルコアカウント専用になる。

既存アカウントの国を変更することも理論上可能だが、購入済みコンテンツや App が移行できないリスクがある。新規作成を強く推奨。

VPN でトルコに接続する（推奨）

サインアップ時の IP アドレスをトルコにしておくと、地域判定がスムーズになる。無料の ProtonVPN や Windscribe でもトルコサーバーを選べる。

appleid.apple.com で新規 Apple ID を作成

メールアドレスとパスワードを入力後、国/地域を「トルコ」に設定する。住所の入力が必要になるので、以下のようなトルコの架空住所を入力する。

住所例：Bağcılar Mahallesi, Atatürk Caddesi No:1, Bağcılar, İstanbul, 34200, TR
電話番号：+90 から始まるダミー番号（例：+90 555 000 0000）

支払い方法を設定する

ここが最大のポイント。トルコ App Store で支払うには、以下のいずれかの方法が現実的だ。

方法A：ギフトカードを使う（最も簡単）
eBay や Eneba、G2A でトルコの App Store ギフトカード（TRY建て）を購入できる。クレジットカード不要。金額チャージ制なので管理しやすい。

方法B：Wise のバーチャルカードを使う
Wise（旧TransferWise）で TRY に両替し、バーチャルデビットカードで支払う。直接登録できないケースもあるので、ギフトカードのほうが確実。

日本のクレジットカードをトルコアカウントに直接登録するのは難しい場合が多い。ギフトカード購入が最も安定している。

iPhone / Mac でアカウントを切り替える

iCloud の機能（写真、バックアップ等）を利用したい場合は、「設定 → Apple ID → iCloud」だけトルコアカウントにサインインする方法がある。App の購入は日本アカウントのまま維持できる。

手順：設定 → 一番上の名前 → サインアウト（iCloudのみ） → トルコアカウントでサインイン

サブスクリプションを購入する

トルコアカウントでサインインした状態で、App Store → アカウント → サブスクリプション からプランを選択。または設定 → Apple ID → iCloud+ から直接申込める。

リスクと注意点

アカウント停止リスク

Apple の利用規約では「居住国の App Store を使用すること」と定めている。 トルコアカウントを使い続けることは規約違反にあたる可能性がある。 実際に停止されたという報告は少ないが、ゼロではない。 大切なデータや購入済みコンテンツをこのアカウントに集めないこと。

為替変動リスク

トルコリラは不安定な通貨だ。Apple が現地価格を改定するタイミングもある。 半年〜1年ごとに節約額を再試算することを推奨する。

ファミリー共有はできない

日本アカウントとトルコアカウントをまたいだファミリー共有は機能しない。 家族と共有している場合は注意が必要。

SUMMARY: ギフトカード購入さえクリアすれば、iCloud 2TB が月 ¥180 程度で使える。日本のアカウントと用途を分けて管理するのがリスク最小化の鍵。
